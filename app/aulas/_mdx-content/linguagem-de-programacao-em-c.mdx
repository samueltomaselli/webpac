# Linguagem de programação em C

## O que é?

A linguagem de programação C é uma das mais antigas e influentes linguagens de programação ainda em uso. Ela foi desenvolvida no início dos anos 1970 por Dennis Ritchie na Bell Labs. C é conhecida por sua eficiência, controle sobre o hardware e ampla aplicação, desde sistemas operacionais até software de aplicação.

## Estrutura de um Programa em C

```
#include <stdio.h>
int  main() {
printf("Hello, World!\n"); return  0; }
```

## Conceitos Fundamentais em C

**Variáveis e Tipos de Dados:**

C suporta vários tipos de dados básicos, incluindo:

- `int`: Números inteiros.
- `float`: Números de ponto flutuante de precisão simples.
- `double`: Números de ponto flutuante de precisão dupla.
- `char`: Caracteres individuais.

## Conclusão

A linguagem de programação C é fundamental tanto para iniciantes quanto para programadores experientes por sua eficiência e controle sobre o hardware. Conhecer C ajuda a entender os fundamentos da computação e serve como base para aprender outras linguagens de programação.
