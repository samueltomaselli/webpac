# Lógica de programação

## O que é lógica de programação?

A lógica de programação é um conjunto de princípios e técnicas que orientam o desenvolvimento de algoritmos e a solução de problemas computacionais. Trata-se de uma forma de raciocínio que envolve a formulação de um passo a passo detalhado para atingir um objetivo específico em um programa de computador.

## Estrutura de dados

A lógica de programação utiliza estruturas de controle para determinar o fluxo de execução de um programa. As principais estruturas de controle incluem:

- **Sequência:** Execução das instruções na ordem em que aparecem.
- **Decisão:** Uso de instruções condicionais (`if`, `else`) para executar diferentes partes do código com base em condições específicas.
- **Repetição:** Laços (`for`, `while`) que repetem um bloco de instruções várias vezes até que uma condição seja satisfeita.

## Algoritmos

Um algoritmo é basicamente um conjunto de instruções ou passos que você segue para realizar uma tarefa específica. Vamos imaginar que você está ensinando alguém a fazer um sanduíche. O algoritmo para fazer um sanduíche poderia ser algo assim:

1.  **Pegue dois pedaços de pão.**
2.  **Passe manteiga no pão.**
3.  **Coloque uma fatia de queijo em um dos pedaços de pão.**
4.  **Coloque uma fatia de presunto em cima do queijo.**
5.  **Junte os dois pedaços de pão.**

## Modularidade

A divisão de um programa em módulos ou funções menores e reutilizáveis é um princípio essencial da lógica de programação. Isso facilita a compreensão, a manutenção e o desenvolvimento do código.

## **Pensamento Algorítmico**

Desenvolver a lógica de programação envolve pensar de maneira algorítmica, ou seja, ser capaz de decompor um problema em etapas menores e criar uma solução passo a passo.

                                   Exemplo Prático:

Vamos considerar um exemplo simples de lógica de programação: calcular a soma de todos os números de 1 a 10.

1.  **Definição do problema:** Queremos calcular a soma de todos os números inteiros de 1 a 10.
2.  **Desenvolvimento do algoritmo:**
    - Inicializar uma variável `soma` com 0.
    - Para cada número `i` de 1 a 10, adicionar `i` à variável `soma`.
    - Exibir o valor da variável `soma`.

## Exercícios

### Exercício 1

Escreva um algoritmo que:

1. Pergunte o nome de uma pessoa;
2. Pergunte qual o sobrenome da pessoa;
3. Mostre o nome completo na tela.

### Exercício 2

Escreva um algoritmo que leia dois valores numéricos e depois mostre a diferença entre eles (o primeiro menos o segundo).

### Exercício 3

Crie um algoritmo que leia dois números nas variáveis numA e numB, nessa ordem, e os mostre na tela em ordem inversa, isto é, se os dados lidos forem 7 e 15, por exemplo, devem ser mostrados na ordem 15 e 7, um embaixo do outro (em linhas distintas).
