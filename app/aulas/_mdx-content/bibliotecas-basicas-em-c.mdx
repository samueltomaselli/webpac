# Bibliotecas básicas em C

## Funcionalidade

Essas bibliotecas básicas fornecem as ferramentas essenciais para realizar operações comuns na programação em C. Elas facilitam a manipulação de dados, a realização de cálculos, a leitura e escrita de dados, e a interação com o sistema operacional, tornando a programação mais eficiente e menos propensa a erros. Conhecê-las e saber como usá-las é fundamental para qualquer programador que esteja aprendendo ou trabalhando com C.

## Exemplo: stdlib.h (Standard Library)

Serve para alocação de memória, controle de processos, conversões e outras utilidades gerais.

- **Funções Comuns:**
  - `malloc`: Aloca memória dinamicamente.
  - `free`: Libera memória alocada dinamicamente.
  - `exit`: Termina o programa.
  - `atoi`: Converte uma string para um inteiro.
  - `rand`: Gera um número aleatório.
  - `srand`: Inicializa o gerador de números aleatórios.

```c

    #include <stdlib.h>
    #include <stdio.h>

    int main() {
        int *array = (int *)malloc(5 * sizeof(int));
        if (array == NULL) {
            printf("Erro na alocação de memória.\n");
            exit(1);
        }

        for (int i = 0; i < 5; i++) {
            array[i] = i * i;
        }

        for (int i = 0; i < 5; i++) {
            printf("%d ", array[i]);
        }
        printf("\n");

        free(array);
        return 0;
    }

```

## Exemplo: string.h (String Handling)

Serve para manipulação de strings (cadeias de caracteres).

- **Funções Comuns:**
  - `strlen`: Retorna o comprimento de uma string.
  - `strcpy`: Copia uma string para outra.
  - `strcat`: Concatena (junta) duas strings.
  - `strcmp`: Compara duas strings.
  - `strchr`: Encontra a primeira ocorrência de um caractere em uma string.
  - `strstr`: Encontra a primeira ocorrência de uma substring em uma string.

**Exemplo:**

```c

    #include <string.h>
    #include <stdio.h>

    int main() {
        char str1[20] = "Hello";
        char str2[20] = "World";

        strcat(str1, " ");
        strcat(str1, str2);

        printf("Concatenated String: %s\n", str1);
        printf("Length of String: %lu\n", strlen(str1));

        return 0;
    }


```

## Exemplo: math.h (Mathematical Functions)

Serve para realizar operações matemáticas comuns.

- **Funções Comuns:**
  - `sqrt`: Calcula a raiz quadrada de um número.
  - `pow`: Calcula a potência de um número.
  - `sin`: Calcula o seno de um ângulo (em radianos).
  - `cos`: Calcula o cosseno de um ângulo (em radianos).
  - `tan`: Calcula a tangente de um ângulo (em radianos).
  - `log`: Calcula o logaritmo natural de um número.
  - `exp`: Calcula o valor de `e` elevado a um número.

**Exemplo:**

```c
#include <math.h>
#include <stdio.h>

int main() {
    double num = 16.0;
    double raiz_quadrada = sqrt(num);
    double base = 2.0;
    double expoente = 3.0;
    double potencia = pow(base, expoente);
    double angulo = M_PI / 4; // 45 graus em radianos
    double seno = sin(angulo);
    double cosseno = cos(angulo);
    double tangente = tan(angulo);

    printf("Raiz quadrada de %.2f: %.2f\n", num, raiz_quadrada);
    printf("%.2f elevado a %.2f: %.2f\n", base, expoente, potencia);
    printf("Seno de 45 graus: %.2f\n", seno);
    printf("Cosseno de 45 graus: %.2f\n", cosseno);
    printf("Tangente de 45 graus: %.2f\n", tangente);

    return 0;
}

```
